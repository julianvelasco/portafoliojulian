# Portafolio Minimalista - Julián Velasco

Este es un portafolio de alto nivel diseñado para **Julián Velasco**, Ingeniero de Sistemas. El diseño es minimalista, moderno y premium, con un enfoque en la experiencia de usuario (UX) y una estética tecnológica limpia.

## 🚀 Características

- **Diseño Premium:** Estética oscura con acentos en azul/teal.
- **Interactividad:** Animaciones suaves, efectos de hover avanzados y transiciones fluidas.
- **Lightbox de Proyectos:** Galería integrada para visualizar proyectos en pantalla completa con fondo desenfocado.
- **Optimizado para IA:** Secciones dedicadas a habilidades en IA, LLMs y Machine Learning.
- **Fondo Dinámico:** Uso de videos de fondo sutiles para una sensación de profundidad y modernidad.
- **Totalmente Responsivo:** Adaptado para dispositivos móviles, tablets y escritorio.

## 🛠️ Tecnologías

- **HTML5:** Estructura semántica.
- **CSS3:** Diseño personalizado (Vanilla CSS) con variables, Flexbox, Grid y animaciones.
- **JavaScript:** Interactividad personalizada sin dependencias externas.
- **SVGs:** Iconografía vectorial de alta calidad.

## 📂 Estructura del Proyecto

- `index.html`: Estructura principal de la página.
- `style.css`: Todos los estilos y el sistema de diseño.
- `main.js`: Lógica de interactividad, animaciones y modal.
- `assets/`: Contiene videos de fondo, imágenes de proyectos y logos.

## 🔧 Instalación Local

1. Clona este repositorio o descarga el archivo ZIP.
2. Abre el archivo `index.html` en tu navegador favorito.
3. (Opcional) Para una mejor experiencia con los videos de fondo, sirve los archivos usando un servidor local (ej. `npx serve` o la extensión Live Server de VS Code).
